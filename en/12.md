```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Spiral SVG with Arrows</title>
    <style>
        /*svg {
            border: 1px solid black;
        }*/
    </style>
</head>
<body>
    <svg id="spiral" width="800" height="800" xmlns="http://www.w3.org/2000/svg">
        <defs>
            <marker id="arrow-black" markerWidth="10" markerHeight="10" refX="5" refY="5"
                    orient="auto-start-reverse" markerUnits="strokeWidth">
                <path d="M 0 0 L 10 5 L 0 10 z" fill="black"/>
            </marker>
            <marker id="arrow-blue" markerWidth="5" markerHeight="5" refX="2.5" refY="2.5"
                    orient="auto-start-reverse" markerUnits="strokeWidth">
                <path d="M 0 0 L 5 2.5 L 0 5 z" fill="blue"/>
            </marker>
        </defs>
    </svg>
    <script>
        function createSpiralPath(cx, cy, turns, spacing) {
            let pathData = `M ${cx},${cy}`;
            const points = 100; // Number of points per turn

            for (let i = 0; i < turns * points; i++) {
                const angle = i * (2 * Math.PI / points); // Positive angle for clockwise
                const radius = spacing * angle / (2 * Math.PI);
                const x = cx + radius * Math.cos(angle);
                const y = cy + radius * Math.sin(angle);
                pathData += ` L ${x},${y}`;
            }

            return pathData;
        }

        function createCurvedLine(cx, cy, length) {
            // Create a random curved line path
            const xEnd = cx + length;
            let pathData = `M ${cx},${cy}`;
            
            for (let i = 0; i <= length; i += length / 10) {
                const x = cx + i;
                const y = cy - i + (Math.random() - 0.5) * 50; // Create a random offset for y, extending upwards
                pathData += ` L ${x},${y}`;
            }

            return pathData;
        }

        const svg = document.getElementById('spiral');
        const spiralPathData = createSpiralPath(400, 400, 20, 10);
        const curvedLinePathData = createCurvedLine(400, 400, 200); // Shorter length for the curved line

        const spiralPath = document.createElementNS('http://www.w3.org/2000/svg', 'path');
        spiralPath.setAttribute('d', spiralPathData);
        spiralPath.setAttribute('stroke', 'black');
        spiralPath.setAttribute('fill', 'none');
        spiralPath.setAttribute('marker-end', 'url(#arrow-black)'); // Add black arrow to the end

        const curvedLinePath = document.createElementNS('http://www.w3.org/2000/svg', 'path');
        curvedLinePath.setAttribute('d', curvedLinePathData);
        curvedLinePath.setAttribute('stroke', 'blue');
        curvedLinePath.setAttribute('fill', 'none');
        curvedLinePath.setAttribute('stroke-width', '2'); // Set the line to be thinner
        curvedLinePath.setAttribute('marker-end', 'url(#arrow-blue)'); // Add blue arrow to the end

        svg.appendChild(spiralPath);
        svg.appendChild(curvedLinePath);
    </script>
</body>
</html>
```

## 12.6. Tweaking Your Brain's Safety Threshold

There's a ton of theories and research on why kids seem to learn faster and better. The most convincing one, in my words, is this:

> For kids, it's not just learning — it's a **struggle for survival**.

Our brains are obsessed with saving energy and terrified of dying. That's because the brain's main job is to keep the body safe. After all, the brain only matters if the rest of the body is functioning.

So, your brain won't let you work beyond your limits. Just like a car engine, if it runs at 100% capacity, it won't last long before it breaks down. Engineers set a safety threshold, like capping the engine's power at 70% of its max. This keeps things safe and stable over the long haul.

Why do kids seem to "learn everything"? Because from their perspective, everything is essential for survival. If adults can do it, they must too. Simple as that. At that age, their brains set high thresholds for everything, almost always aiming for 100%.

Why do we lose that drive as we grow up? Because our brains have repeatedly confirmed, "Even if I don't get this, life goes on. I can eat, sleep, and survive just fine." So, the brain lowers the safety threshold from 90% or 85% to 60% or 50%, maybe even less.

Often, whether someone learns or masters something has nothing to do with IQ. If there's any cause-and-effect relationship, it's the opposite of what many think. IQ isn't the "cause," it's the "effect." When the brain sets a low threshold, you quit halfway, and the result is your "IQ" doesn't improve, or relatively speaking, it drops.

Why do so many people read tons of books, hear loads of advice, yet take no action or change? It's tied to the safety threshold their brain "secretly" sets — and they're clueless about it. This ignorance leads to anxiety, leaving them in a corner, pulling their hair, asking, "What's wrong with me?!"

The evolutionary gains from hundreds of thousands of years of genetic inheritance aren't suited for this era of information overload. The brain is a network of neurons, more complex than the galaxy, while knowledge is an ever-expanding network. The world is constantly changing, with trends accelerating and growing. Honestly, the brain struggles to keep up with these changes and their speed.

The least adaptable part might be those default safety thresholds. For hundreds of thousands of years, survival was the issue, so the brain prioritized it above all else. Suddenly, survival isn't a problem. Life is super safe, and the cost of essentials is dropping.

In this scenario, sticking to those ancient safety thresholds means doing nothing — the ultimate "lying flat." Two or three meals a day, a glass of milk, two eggs, and half an apple are enough to survive. Survival isn't an issue and shouldn't be, so why not lie flat?

> Lowering your brain's safety threshold to the minimum...

> Find ways to raise your brain's safety threshold...
